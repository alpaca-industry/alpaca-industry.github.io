애플 문서 RSS (Apple Documentation RSS)

애플 문서는 내용도 좋고, 업데이트도 잘 되는 편인데 검색에도 잘 안나오고, 개발팀에서 홍보에도 별 관심이 없다. (반면 구글 안드로이드 쪽은 따로 개발 유튜브 채널과 블로그도 있지만, 문서 관리가 아주 개판)

매번 들어가서 확인하기는 귀찮아서 찾아보니 몇가지 RSS가 있었다.

Site Changes : https://developer.apple.com/news/site-updates/rss/site-updates.rss
Latest News : https://developer.apple.com/news/rss/news.rss
Software Releases : https://developer.apple.com/news/releases/rss/releases.rss

이것만 봐도 최신 정보는 놓치지 않고 볼 수 있음.
